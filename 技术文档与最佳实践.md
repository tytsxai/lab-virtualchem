# VirtualChemLab 技术文档与最佳实践

## 📚 项目架构文档

### 核心架构概述

VirtualChemLab 采用分层架构设计，主要包含以下层次：

```
┌─────────────────────────────────────────┐
│                 UI Layer                │
│  (PySide6, 用户界面, 交互组件)           │
├─────────────────────────────────────────┤
│              Service Layer              │
│  (业务逻辑, 服务接口, 工作流管理)         │
├─────────────────────────────────────────┤
│               Core Layer                │
│  (核心功能, 依赖注入, 事件总线)           │
├─────────────────────────────────────────┤
│             Infrastructure              │
│  (存储, 缓存, 监控, 安全)               │
└─────────────────────────────────────────┘
```

### 核心组件说明

#### 1. 依赖注入容器 (DIContainer)

- **位置**: `src/core/di_container.py`
- **功能**: 管理服务生命周期和依赖关系
- **特性**: 支持单例、瞬态、作用域生命周期
- **使用示例**:

```python
from src.core.di_container import DIContainer, Lifetime

container = DIContainer()
container.register(ICacheManager, CacheManager, lifetime=Lifetime.SINGLETON)
cache = container.resolve(ICacheManager)
```

#### 2. 事件总线 (EventBus)

- **位置**: `src/core/event_bus.py`
- **功能**: 实现组件间松耦合通信
- **特性**: 支持同步/异步事件、优先级、过滤
- **使用示例**:

```python
from src.core.event_bus import EventBus, Event

event_bus = EventBus()
event_bus.subscribe("experiment.started", handler_function)
event_bus.publish(Event(name="experiment.started", data={"id": "exp_001"}))
```

#### 3. 配置管理 (ConfigLoader)

- **位置**: `src/core/config_loader.py`
- **功能**: 统一配置加载和验证
- **特性**: 多环境支持、环境变量覆盖、Pydantic验证
- **使用示例**:

```python
from src.core.config_loader import get_config

config = get_config()
database_url = config.database.url
```

#### 4. 内存管理 (MemoryManager)

- **位置**: `src/core/memory_manager.py`
- **功能**: 内存使用监控和泄漏检测
- **特性**: 弱引用跟踪、自动清理、泄漏检测
- **使用示例**:

```python
from src.core.memory_manager import memory_manager

memory_manager.track_object(my_object, "experiment_data")
memory_manager.cleanup_memory()
```

#### 5. 缓存管理 (CacheManager)

- **位置**: `src/core/cache_manager.py`
- **功能**: 智能缓存管理
- **特性**: 多种策略、TTL支持、Redis集成
- **使用示例**:

```python
from src.core.cache_manager import get_cache_manager

cache = get_cache_manager()
cache.set("key", "value", ttl=3600)
value = cache.get("key")
```

### 4. 线程安全规范

#### 使用锁保护共享资源

```python
import threading

class ThreadSafeCounter:
    def __init__(self):
        self._value = 0
        self._lock = threading.RLock()

    def increment(self):
        with self._lock:
            self._value += 1

    def get_value(self):
        with self._lock:
            return self._value
```

#### 避免死锁

```python
# 好的做法：按固定顺序获取锁
def transfer_money(account1, account2, amount):
    if account1.id < account2.id:
        with account1.lock:
            with account2.lock:
                # 执行转账
                pass
    else:
        with account2.lock:
            with account1.lock:
                # 执行转账
                pass
```

## 🔒 安全指南

### 1. 输入验证

#### 数据验证

```python
from pydantic import BaseModel, validator

class ExperimentData(BaseModel):
    name: str
    description: str
    parameters: dict

    @validator('name')
    def validate_name(cls, v):
        if len(v) > 100:
            raise ValueError('实验名称不能超过100字符')
        return v

    @validator('parameters')
    def validate_parameters(cls, v):
        if not isinstance(v, dict):
            raise ValueError('参数必须是字典类型')
        return v
```

#### SQL注入防护

```python
# 好的做法：使用参数化查询
def get_experiment_by_id(exp_id: str):
    query = "SELECT * FROM experiments WHERE id = :exp_id"
    return session.execute(query, {"exp_id": exp_id})

# 避免的做法：字符串拼接
def get_experiment_by_id(exp_id: str):
    query = f"SELECT * FROM experiments WHERE id = '{exp_id}'"  # 危险！
    return session.execute(query)
```
