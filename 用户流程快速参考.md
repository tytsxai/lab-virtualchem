# 用户流程快速参考

本页提供 VirtualChemLab 的高频操作路径速查，适合作为“第一次用完以后”的记忆卡片。

> 注：不同版本 UI 文案可能略有差异；若以 UI 为准的详尽说明请阅读 `docs/USER_MANUAL.md`。

## 1. 典型流程

1. 启动应用：`python3 main.py`（开发：`python3 main.py --env development`）
2. 选择实验模板（或从模板管理创建/导入）
3. 按步骤提示完成实验操作
4. 查看结果与评分/记录
5. 导出/生成报告（若启用 PDF 导出插件）

## 2. 常用位置

- 故障排查：`docs/TROUBLESHOOTING.md`
- 用户手册：`docs/USER_MANUAL.md`
