# 开发者面板快速参考

本页是 `tools/developer_panel.py` 的“速查卡”，用于快速记忆入口与常用操作。

## 启动

```bash
python3 tools/developer_panel.py
```

如果你在虚拟环境里（推荐）：

```bash
venv311/bin/python tools/developer_panel.py
```

## 常用入口（建议）

- 启动主应用（开发模式）：`python3 main.py --env development`
- 快速测试（跳过覆盖率门禁）：`make test-fast`
- 代码检查：`make lint`
- 自动修复（Ruff）：`make lint-fix`
- 格式化：`make format`

## 排障提示

- 无显示环境/CI 下 Qt 相关测试建议使用：`QT_QPA_PLATFORM=offscreen`
- 日志与报告通常输出到：`logs/`、`reports/`、`htmlcov/`

详细说明参见：`docs/DEVELOPER_PANEL.md`、`docs/TROUBLESHOOTING.md`。
