# 虚拟化学实验室 - 菜单功能说明

## 📋 功能概述

本文档列出了虚拟化学实验室主窗口的所有菜单功能及其快捷键。

---

## 🗂️ 菜单结构

### 1. 文件菜单

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 新建实验 | `Ctrl+N` | 创建新的实验 |
| 打开记录 | `Ctrl+O` | 打开历史实验记录 |
| 保存实验进度 | `Ctrl+S` | 保存当前实验的进度 |
| 导入数据... | - | 从JSON文件导入实验数据 |
| 导出数据... | `Ctrl+E` | 导出实验数据到JSON文件 |
| 退出 | `Alt+F4` | 退出应用程序 |

### 2. 编辑菜单

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 撤销 | `Ctrl+Z` | 撤销上一步操作（开发中） |

| 查找... | `Ctrl+F` | 查找内容（开发中） |
| 偏好设置... | `Ctrl+,` | 打开设置对话框 |

## 🔧 实现的功能处理器

所有菜单项都已实现对应的处理函数：

✅ **文件操作**
- `on_save_progress()` - 保存实验进度
- `on_import_data()` - 导入数据
- `on_export_data()` - 导出数据

✅ **编辑操作**
- `on_undo()` - 撤销（待实现完整功能）
- `on_redo()` - 重做（待实现完整功能）
- `on_find()` - 查找（待实现完整功能）

✅ **实验操作**
- `on_pause_experiment()` - 暂停实验
- `on_batch_export()` - 批量导出报告
- `on_validate_data()` - 验证实验数据

✅ **视图操作**
- `on_toggle_fullscreen()` - 切换全屏
- `on_toggle_toolbar()` - 切换工具栏
- `on_toggle_statusbar()` - 切换状态栏
- `on_switch_layout()` - 切换布局

✅ **工具操作**
- `on_backup_data()` - 备份数据
- `on_restore_data()` - 恢复数据
- `on_validate_templates()` - 验证模板
- `on_clear_cache()` - 清除缓存

✅ **数据分析**
- `on_show_statistics()` - 显示统计
- `on_compare_experiments()` - 对比实验（待实现）
- `on_show_trends()` - 趋势分析（待实现）

✅ **窗口管理**
- `on_toggle_maximize()` - 切换最大化
- `on_close_all_windows()` - 关闭所有窗口

✅ **帮助功能**
- `on_show_shortcuts()` - 显示快捷键
- `on_show_tutorial()` - 视频教程
- `on_open_online_docs()` - 在线文档
- `on_send_feedback()` - 发送反馈
- `on_check_updates()` - 检查更新

## 📝 开发说明

### 添加新菜单项
1. 在 `create_menu_bar()` 方法中添加菜单项
2. 设置快捷键（如需要）
3. 创建对应的处理函数 `on_xxx()`
4. 更新本文档

### 测试菜单功能
运行测试脚本验证所有菜单和处理器：
