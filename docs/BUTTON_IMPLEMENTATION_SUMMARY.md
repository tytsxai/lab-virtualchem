### 7. 化学反应动力学模拟 ✅

**文件**: `src/ui/chemical_reaction_simulator.py`
**问题**: `_update_reactions()` 方法为空，反应无法进行
**解决**:

- 实现了酸碱中和反应（HCl + NaOH → NaCl + H2O）的动力学模拟
- 计算反应速率（与浓度乘积成正比）
- 更新试剂量和生成产物
- 实时计算并更新pH值
- 发送 `reaction_completed` 信号

**影响**: 虚拟实验中的化学反应可以真实模拟，pH值会随反应进行变化

**关键逻辑**:

```python
# 反应速率计算
reaction_amount = min(hcl_amount, naoh_amount) * 0.01  # 每次消耗1%

# pH值计算
if reagents["hcl"] > reagents["naoh"]:
    new_ph = 3.0 + (reagents["naoh"] / reagents["hcl"]) * 4.0
elif reagents["naoh"] > reagents["hcl"]:
    new_ph = 11.0 - (reagents["hcl"] / reagents["naoh"]) * 4.0
else:
    new_ph = 7.0
```
