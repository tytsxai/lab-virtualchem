## 概述

本指南详细描述了 VirtualChemLab 的完整用户操作流程，确保用户能够顺畅地使用所有功能。

### 设计原则

- **简单直观**: 减少学习曲线，让用户快速上手
- **流程清晰**: 每个步骤都有明确的目标和反馈
- **灵活可选**: 支持多种使用路径和个性化配置
- **容错性强**: 提供充分的引导和错误恢复机制

## 用户角色

### 1. 学生用户 (Student)

- **主要目标**: 完成实验学习，提升化学知识
- **使用场景**: 课堂练习、自主学习、考试测验
- **权限级别**: 标准

### 2. 访客用户 (Guest)

- **主要目标**: 体验功能、了解产品
- **使用场景**: 试用、演示
- **权限级别**: 受限

## 详细流程说明

### 阶段一：应用启动

#### 1.1 启动检查

**自动执行的检查项：**

- ✅ Python 环境检查（3.10+）
- ✅ 依赖库完整性验证
- ✅ 配置文件加载
- ✅ 必要目录创建
- ✅ 实验模板扫描

**可能的问题及解决：**

```python
问题：缺少依赖库
解决：自动提示安装命令
     pip install -r requirements.txt

问题：配置文件损坏
解决：自动使用默认配置并提示用户

问题：实验模板缺失
解决：提示下载示例模板或创建新模板
```

#### 1.2 首次使用向导

**适用对象：** 首次启动或重置后的用户

**向导内容：**

```
第1页：欢迎页面
- 欢迎词和产品介绍
- 快速功能概览
- 版本信息

第2页：基本功能介绍
- 实验系统说明
- 游戏化功能
- 成就系统
- 知识库

第3页：操作指南
- 界面导航
- 基本操作
- 快捷键列表

第4页：使用技巧
- 实验技巧
- 性能优化
- 安全注意事项

第5页：完成设置
- 选择主题
- 选择语言
- 启用功能选项
  □ 显示操作提示
  □ 启用成就系统
  □ 自动保存进度
  □ 启用游戏模式
```

**用户操作：**

- 可以逐页浏览学习
- 可以跳过向导
- 可以选择"不再显示"

#### 1.3 用户身份确认

**简化模式（默认）：**

```

无需登录，直接使用
```

**完整模式（可选）：**

```
1. 输入用户名/学号
2. 进入系统

#### 3.2 步骤类型和操作

**类型1：确认型检查点**

```
界面显示:
□ 我已穿戴护目镜
□ 我已穿戴实验服
□ 我已了解实验风险

操作: 勾选所有选项 → 点击"提交"

验证: 检查是否全部勾选

反馈:
✓ 通过: "安全准备完成，可以开始实验"
✗ 失败: "请确认完成所有安全准备"
```

**类型2：数值输入**

```
界面显示:
请输入移取的HCl体积: [_______] mL

操作: 输入数值 → 点击"提交"

验证规则:
• 类型检查: 必须是数字
• 范围检查: 24.5 ≤ 输入值 ≤ 25.5
• 精度检查: 小数点后最多2位

反馈:
✓ 通过: "体积准确，继续下一步"
✗ 失败: "体积偏差过大，期望: 25.0±0.5 mL，实际: XX mL"
      "请重新操作"
```

**类型3：单选**

```
界面显示:
选择合适的指示剂:
○ 酚酞
○ 甲基橙
○ 石蕊试纸

操作: 选择一个选项 → 点击"提交"

验证: 检查是否为正确选项

反馈:
✓ 通过: "选择正确！酚酞适用于强碱滴定强酸"
✗ 失败: "选择不当，请查看[知识点]了解指示剂选择原则"
```

**类型4：多选**

```
界面显示:
选择需要的安全防护装备(可多选):
□ 护目镜 ✓必需
□ 实验服 ✓必需
□ 手套   ✓必需
□ 口罩

操作: 勾选多个选项 → 点击"提交"

验证: 检查必需项是否全部选中

反馈:
✓ 通过: "安全装备齐全"
✗ 失败: "缺少必需装备: 护目镜、手套"
```

**类型5：排序**

```
界面显示:
请按正确顺序排列实验步骤(拖动调整):

[1] 准备试剂        ⋮
[2] 移取待测液      ⋮
[3] 加入指示剂      ⋮
[4] 滴定至终点      ⋮

操作: 拖动排序 → 点击"提交"

验证: 检查顺序是否正确

反馈:
✓ 通过: "步骤顺序正确"
✗ 失败: "顺序有误，请重新排列"
      "提示: 应该先加入指示剂再滴定"
```

**类型6：交互式操作**

```
界面显示:

• 烧杯

操作:
1. 拖动器材到工作台
2. 点击器材进行操作
3. 观察实验现象

验证:
• 检测操作序列
• 验证最终状态
• 评分操作规范性

反馈:
✓ 通过: "操作规范，实验成功"
✗ 失败: "操作不当，请重试"
      "错误: 未进行移液管润洗"

## 常见问题处理

### Q1: 启动失败

**症状:** 双击启动脚本后闪退或报错

**解决步骤:**

```
1. 检查Python版本
   python --version
   → 需要 Python 3.10+

2. 检查依赖安装
   pip install -r requirements.txt

3. 检查配置文件
   → 删除config.json让系统重新生成

4. 查看日志
   → 打开 logs/app.log 查看详细错误
```

### Q2: 实验无法提交

**症状:** 点击提交按钮无反应

**可能原因:**

```
1. 未填写必填项
   → 检查是否有红色标记的必填字段

2. 输入格式错误
   → 数值字段中输入了文字

3. 网络连接问题(在线模式)
   → 切换到离线模式

4. 系统卡顿
   → 查看任务管理器，关闭其他程序
```

### Q3: 数据丢失

**症状:** 实验记录看不到或消失

**恢复方法:**

```
1. 检查数据目录
   → data/records/ 目录是否存在

2. 从备份恢复
   [工具] → [恢复数据]
   → 选择最近的备份文件

3. 检查自动保存设置
   [编辑] → [偏好设置] → [行为]
   → 确保"自动保存进度"已勾选
```

### Q4: 性能问题

**症状:** 界面卡顿、反应慢

**优化方法:**

```
1. 降低图形质量
   [编辑] → [偏好设置] → [性能]
   → 渲染质量: 低
   → 最大帧率: 30

2. 关闭游戏模式
   → 游戏模式会消耗更多资源

3. 关闭粒子效果
   实验界面 → [⚙ 设置]
   → 粒子效果: 关闭

4. 清理缓存
   [工具] → [清除缓存]
```
