## 🆘 获取帮助

### 常见问题

1. **找不到某个功能的文档？**
   - 使用本索引的搜索功能（Ctrl+F）
   - 查看 [features.md](features.md) 功能列表
   - 查看 [API_REFERENCE.md](API_REFERENCE.md) API参考

2. **文档版本不匹配？**
   - 检查文档顶部的版本号
   - 查看 [CHANGELOG.md](../CHANGELOG.md) 了解变更
   - 参考最新的项目版本

3. **需要示例代码？**
   - 查看各文档的"示例"章节
   - 参考 `examples/` 目录
   - 查看单元测试代码
