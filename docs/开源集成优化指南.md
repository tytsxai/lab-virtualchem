## 🔧 问题修复

### 常见问题

#### 1. 核心依赖缺失
```bash
# 症状: 应用无法启动
# 解决:
pip install -r requirements.txt

# 验证
python -c "import PySide6, numpy, scipy; print('✅')"
```

#### 2. 版本冲突
```bash
# 症状: 导入错误或功能异常
# 解决:
pip install --upgrade -r requirements.txt

# 检查冲突
python scripts/check_integration_health.py
```

#### 3. 插件加载失败
```python
# 症状: 功能降级或警告
# 解决:
from src.plugins import registry

# 查看插件状态
registry.print_health_report()

# 安装缺失插件
# pip install [plugin_name]
```

#### 4. 已知冲突

**matplotlib vs pyqtgraph**
- **问题**: Qt后端可能冲突
- **解决**: 使用插件系统隔离，PyQtGraph作为可选插件

**reportlab vs weasyprint**
- **问题**: PDF生成依赖冲突
- **解决**: 智能选择可用的库，互为备份

## 📈 性能优化

### 1. 懒加载插件
```python

@lazy_load
```

## 💡 贡献指南

### 添加新的开源库

1. **评估许可证兼容性**
   ```python
   # 检查许可证
   # MIT/BSD/Apache-2.0: 可直接集成
   # LGPL: 需动态链接隔离
   # GPL: 必须作为独立插件
   ```

2. **选择依赖文件**
   - 核心功能 → `requirements.txt`
   - 可选功能 → `requirements-optional.txt`
   - 增强功能 → `requirements-enhanced.txt`

3. **注册插件 (如果是可选)**
   ```python
   from src.plugins import registry

   registry.register(
       name="new_plugin",
       description="功能描述",
       module_name="package_name",
       license="许可证",
       fallback=fallback_function,
       dependencies=["dep1", "dep2"],
       conflicts_with=["conflict_lib"]
   )
   ```

4. **更新文档**
   - 在本文档中添加说明
   - 更新README.md
   - 运行健康检查验证
