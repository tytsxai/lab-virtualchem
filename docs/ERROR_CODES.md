# Error Codes（错误码完整列表）

本文件用于集中列出错误码及其含义，方便前后端联调与用户支持。

> 当前仓库的错误码规则与响应格式请先参考：`docs/API_EVENT_PROTOCOL.md` 的“错误码体系”章节。

## 规则（摘要）

- 错误码应可分类、可检索，并与用户可读消息解耦
- 对外暴露的错误信息需避免泄漏敏感细节
