## 修复方法

### 1. 使用logger替代print

```python
# 不好的方式
print(f"用户登录: {username}")

# 推荐方式
from src.utils.logger import get_logger
logger = get_logger(__name__)
logger.info("用户登录", extra={"username": username})
```

### 2. 批量替换正则表达式

```regex
查找: print\((.*)\)
替换: logger.info(\1)
```

### 3. 需要手动审查的情况

- 调试信息 → logger.debug()
- 错误信息 → logger.error()
- 警告信息 → logger.warning()
- 用户输出 → 保留print或使用UI提示
