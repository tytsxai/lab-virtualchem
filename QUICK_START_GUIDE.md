## 📦 安装

### 1. 克隆或下载项目

```bash

```

### 2. 安装依赖

```bash

```

### 3. 首次启动


## 💡 核心操作

### 开始一个实验

#### 方法1：双击实验

1. 在左侧实验列表中找到目标实验
2. 双击实验名称
3. 实验自动开始 ✅

#### 方法2：使用按钮

1. 单击选中实验
2. 点击"开始实验"按钮
3. 实验开始 ✅

#### 方法3：快捷键

1. 使用方向键选择实验
2. 按 `Enter` 键开始
3. 实验开始 ✅

### 获取帮助

#### 🆘 遇到问题？多种方式获得帮助

1. **按 F1** - 打开完整用户手册
2. **按 Ctrl+Shift+K** - 查看快捷键列表
3. **按 Ctrl+K** - 打开知识库
4. **悬停鼠标** - 查看控件提示
5. **查看底部提示** - 关注快速提示

### 常用快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F1` | 帮助 | 打开用户手册 |
| `Ctrl+N` | 新建 | 开始新实验 |
| `Ctrl+S` | 保存 | 保存实验进度 |
| `Ctrl+Z` | 撤销 | 撤销上一步操作 |
| `Ctrl+Y` | 重做 | 重做已撤销的操作 |
| `Ctrl+K` | 知识库 | 打开化学知识库 |
| `Ctrl+G` | 游戏模式 | 切换游戏模式 |
| `Ctrl+,` | 设置 | 打开偏好设置 |
| `Ctrl+Shift+K` | 快捷键 | 查看所有快捷键 |

## 🔧 遇到问题？

### 启动失败

#### 问题：缺少依赖库

```
❌ 缺少必需的依赖库!
```

**解决方案**：

```bash
pip install -r requirements.txt
```

#### 问题：Python版本过低

```
需要 Python 3.8+，当前为 3.7
```

**解决方案**：

- 升级Python到3.8或更高版本
- 使用虚拟环境管理不同版本
