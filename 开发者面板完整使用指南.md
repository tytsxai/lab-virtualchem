### 启动面板

#### 标准版
```bash

开发者面板.bat

# 方式2：命令行
python tools/developer_panel.py
```

```bash
# 方式1：双击批处理文件
```

- ✅ Python 3.10 或更高版本

- ✅ 所有项目依赖已安装

运行测试脚本验证所有功能：
```bash
python tools/test_all_panel_features.py
```

## 📱 功能详解

### 1. 主应用标签页

#### 🚀 标准启动
- **命令**: `python main.py`
- **功能**: 完整的学生实验模式，包含欢迎向导
- **用途**: 完整功能测试、演示
- **快捷键（增强版）**: 无

#### ⚡ 快速启动
- **命令**: `python main.py --skip-welcome`
- **功能**: 跳过欢迎向导，直接进入实验
- **用途**: 重复测试、快速调试
- **快捷键（增强版）**: `Ctrl+R`

#### 🔥 热加载模式
- **命令**: `python tools/hot_reload_launcher.py`
- **功能**: 监听代码变化，自动重启应用
- **用途**: 开发调试（最推荐）
- **快捷键（增强版）**: `Ctrl+H`
- **监听目录**: `src/`（默认）
- **监听文件**: `.py`, `.qml`
- **防抖时间**: 2秒

**使用技巧：**
- 修改代码后自动重启
- 避免在2秒内多次保存
- 忽略`__pycache__`, `.git`等目录

#### 🔥 热加载+快速启动
- **命令**: `python tools/hot_reload_launcher.py --command "python main.py --skip-welcome"`
- **功能**: 热加载 + 跳过欢迎向导
- **用途**: 最快速的开发方式

#### 🔐 许可证模式启动
- **功能**: 先验证许可证，通过后启动
- **流程**: 
  1. 检查许可证状态
  2. 验证通过 → 启动应用
  3. 验证失败 → 询问是否试用模式

## 🔧 故障排除

### 问题1：面板无法启动

**症状：** 双击bat文件无反应或报错

**解决方案：**
```bash
# 1. 检查Python安装
python --version

# 2. 检查tkinter
python -c "import tkinter"

# 3. 重新安装依赖
pip install -r requirements.txt

# 4. 使用命令行启动查看详细错误
python tools/developer_panel_enhanced.py
```

### 问题2：工具无法运行

**症状：** 点击按钮后提示"文件不存在"

**解决方案：**
```bash
# 1. 运行测试脚本
python tools/test_all_panel_features.py

# 2. 检查tools目录
ls tools/

# 3. 确认工作目录正确
pwd
```

### 问题3：热加载不工作

**症状：** 修改代码后应用不重启

**解决方案：**
```bash
# 1. 检查watchdog安装
pip install watchdog

# 2. 确认监听目录存在
ls src/

# 3. 查看热加载日志
# 日志会显示监听状态

# 4. 手动启动热加载
python tools/hot_reload_launcher.py
```

### 问题4：进程无法停止

**症状：** 点击"停止所有进程"后进程仍在运行

**解决方案：**
```bash
# 1. 使用任务管理器强制结束
# Windows: Ctrl+Shift+Esc → 结束python进程
# Mac: Activity Monitor → 强制退出

# 2. 重启面板
```

### 问题5：日志乱码

**症状：** 日志显示为问号或乱码

**解决方案：**
- 已在代码中修复UTF-8编码
- 如仍有问题，更新到最新版本
- Windows用户确保使用UTF-8编码

### 问题6：许可证验证失败

**症状：** 许可证模式启动失败

**解决方案：**
```bash
# 1. 检查许可证文件
ls data/licenses/

# 2. 运行健康检查
python tools/license_health_check.py

# 3. 重新激活
python tools/license_generator.py activate
```
