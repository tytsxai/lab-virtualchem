## 🔧 核心源码

### 主要模块

```
src/
├── __init__.py                # 包初始化
├── fetcher.py                 # 数据获取 (克隆仓库、列出文件)
├── parsers/                   # 解析器
│   ├── __init__.py
│   ├── experiment_parser.py      # 解析实验代码
│   └── knowledge_parser.py       # 解析知识数据
├── converters/                # 转换器
│   ├── __init__.py
│   ├── template_converter.py     # 转换为实验模板
│   └── card_converter.py         # 转换为知识卡片
└── validators/                # 验证器
    ├── __init__.py
    └── schema_validator.py       # 数据验证
```

### 功能说明

- **fetcher.py**: 负责从 GitHub 获取 chemlab 数据
- **experiment_parser.py**: 解析 Python 示例代码,提取实验信息
- **knowledge_parser.py**: 解析分子数据库,提取化学知识
- **template_converter.py**: 将实验转换为 VirtualChemLab YAML 模板
- **card_converter.py**: 将知识转换为 VirtualChemLab 知识卡片
- **schema_validator.py**: 验证生成的数据是否符合标准

## 🎮 执行脚本

```
scripts/
├── import_all.py              # ⭐ 一键导入所有数据
├── import_experiments.py      # 仅导入实验
├── import_knowledge.py        # 仅导入知识库
└── validate_output.py         # 验证输出数据
```

### 使用说明

```bash
# 导入所有数据
python scripts/import_all.py

# 更新到最新版本
python scripts/import_all.py --update

# 验证数据
python scripts/validate_output.py ../../data/templates
```
